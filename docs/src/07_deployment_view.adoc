[[section-deployment-view]]
== Vista de despliegue

En este caso, habrá un diagrama que represente las relaciones entre elementos que participarán en las acciones de despliegue y cómo se hará para distribuir el proyecto sobre un producto final y la infraestructura para soportarlo.

El diagrama es necesario para entender más a fondo los futuros problemas que pueden aparecer durante el desarrollo de la aplicación, así como para facilitar el mantenimiento de esta.
